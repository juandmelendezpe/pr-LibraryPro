Análisis de Interfaz y Estilos del Frontend

Este documento presenta un desglose de la paleta de colores, la tipografía y los estilos de los componentes principales utilizados en el sistema LiFe System, basado en el archivo de estilos globales.

🎨 Paleta de Colores

El sistema utiliza una paleta moderna, con tonos fríos como principales y un verde esmeralda para destacar elementos secundarios.

🔤 Tipografía

Se utiliza la fuente 'Inter' como tipografía principal. Esta fuente está optimizada para interfaces de usuario, ofreciendo excelente legibilidad en diferentes tamaños de pantalla.

Pila completa: 'Inter', system-ui, -apple-system, sans-serif

🧩 Estilos de Componentes

  • Tarjetas (Cards)
    Utilizan un fondo blanco puro (var(--bg-card)), con bordes redondeados pronunciados de 1rem y una sombra sutil para dar profundidad. Cuentan con un borde tenue (#e2e8f0) para definirlas mejor en fondos claros.
  • Botones (Buttons)
    Los botones tienen bordes redondeados (0.5rem) y no utilizan bordes (border: none). Cuentan con transiciones suaves (0.2s) al pasar el cursor (hover). El botón primario usa el color azul (#2563eb) y se oscurece al interactuar.
  • Barras de Navegación (Sidebar & Nav Links)
    Los enlaces de navegación usan colores apagados que se iluminan al hacer hover, aplicando un sutil fondo blanco translúcido (rgba(255, 255, 255, 0.1)) para indicar interactividad.
  • Scrollbars Personalizados
    Para los contenedores con estilo "Glass", se ocultan los scrollbars del sistema operativo para mantener la ilusión de interfaz inmersiva, reemplazándolos con barras de desplazamiento sutiles y semitransparentes.

✨ Efecto Glassmorphism (Cristal)

Una característica destacada del diseño es el uso del "Glassmorphism", especialmente sobre el fondo estilo escritorio (.desktop-bg). Combina fondos oscuros semitransparentes con desenfoque de fondo (backdrop-filter).
